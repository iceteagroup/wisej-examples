SAP 3D Viewer

Example application to demonstrate some of the features available to the [SAP 3D Visual Enterprise Viewer](https://wiki.scn.sap.com/wiki/display/SVE/SAP+3D+Visual+Enterprise+Viewer) Wisej extension.
